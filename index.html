Vermin is a smart, simple and powerful command line tool for Linux, Windows and macOS. It's designed for developers who want a fresh VM environment with a single command.

It uses VirtualBox to run the VM. Vermin will fetch images on your behalf. Vermin can be used when you need an easy way to obtain a Linux environment up and running in minutes.

For example, If you want to try to install a Kafka cluster, and you need something more than just a docker container.
