completion

stop subcommand used to generates shell completion scripts for Bash, Zsh or PowerShell.

To generate completion script for bash: 

$ vermin completion bash

and to source it use:

$ source <(vermin completion bash)

To generate completion script for zsh: 

$ vermin completion zsh

and to source it use:

$ vermin completion zsh > ~/.zsh/completion/_vermin

To generate completion script for PowerShell: 

$ vermin completion powershell
