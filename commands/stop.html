stop

stop subcommand used to stop one or more VMs.

$ vermin stop vm_01
âœ” Stopping vm_01

Once a VM stopped, you cannot use the command to ssh into the VM.

To stop more than one VM, pass them space separated on the command line: 

$ vermin stop vm_01 vm_02

If you tried to stop an already stopped VM, you will got a message indicating that, as follows: 

$ vermin stop vm_01
VM already stopped, use "vermin start vm_01" to start the VM.

You can make sure the VM is stopped by using the command.

You can always a stopped VM.
