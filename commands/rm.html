rm

When you are done with a VM, you can use the rm subcommand to remove it.

Note: You need to be very careful before execute the rm command, as you cannot undo it. 
$ vermin rm vm_01
✔ Removing vm_01

Once you removed a VM, it is completely deleted and consequencly will be removed from the result of command.

You can remove one or move VMs at once by passing them on the command line: 

$ vermin rm vm_01 vm_02
Force removing VM

By default, you cannot remove a running VM.

$ vermin rm vm_01
Cannot stop running VM, use -f flag to force remove

To remove a running VM you need to pass -f|--force parameter:

$ vermin rm -f vm_01
✔ Stopping vm_01
✔ Removing vm_01
