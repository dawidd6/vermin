start

start subcommand used to start one or more VMs.

$ vermin start vm_01
✔ Starting vm_01
✔ Establishing connection

Then you can use the command to ssh into the VM.

To start more than one VM, pass them space separated on the command line: 

$ vermin start vm_01 vm_02

If you tried to start an already started VM, you will got a message indicating that, as follows: 

$ vermin start vm_01
VM already running, use "vermin ssh vm_01" to ssh into the VM.

You can also or a running VM.

Listing started VMs
We will cover this topic in greater details in the section. but for now you need to know that the ps command by default lists the started and already running VMs.
