ssh

ssh subcommand used to ssh into the VM.

You don't need to provide username or IP address, just provide the VM name

$ vermin ssh vm_01
Welcome to Ubuntu 20.04 LTS (GNU/Linux 5.4.0-26-generic x86_64)
...
...
Last login: Mon Aug 17 00:00:18 2020 from 192.168.100.4
vermin@verminbox:~$

You can exit the VM at any time to return back to the host machine by pressing the CTRL + D. 

vermin@verminbox:~$ logout
Connection to 192.168.100.156 closed.
Sudoers
Inside the VM, user vermin is being used and the password is vermin. but mostly you will never been asked about the password, as the user vermin is an sudoer user.
